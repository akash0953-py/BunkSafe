# 🚀 Overview

Bunksafe is a modern AI-powered attendance management and bunk planning web application designed specifically for college students.

Most students struggle with:

* calculating attendance percentages,
* figuring out how many classes they can safely bunk,
* managing shortage risks,
* tracking subject-wise attendance manually,
* and understanding attendance danger zones before exams.

Bunksafe solves all of this instantly with a clean, fast, mobile-friendly interface.

Instead of spreadsheets and manual calculations, students get:

✅ Real-time attendance analytics
✅ Safe bunk predictions
✅ AI-powered attendance insights
✅ OCR-based attendance extraction
✅ Subject-wise tracking
✅ Smart shortage alerts
✅ Mobile-first experience

---

# ✨ Features

## 🎯 Smart Attendance Dashboard

Track overall attendance percentage across all subjects with a beautiful analytics dashboard.

Features include:

* Overall attendance tracking
* Subject-wise attendance cards
* Attendance trend visualization
* Danger-level indicators
* Minimum attendance goal monitoring

---

## 📉 Safe Bunk Calculator

The core feature of Bunksafe.

Students can instantly know:

* How many classes they can safely bunk
* Which subjects are risky
* Attendance impact after future bunks
* Recovery classes required

No more manual percentage calculations.

---

## ⚠️ Attendance Danger Detection

Bunksafe automatically detects subjects entering risky attendance zones.

### Risk Levels

| Status      | Meaning                                |
| ----------- | -------------------------------------- |
| 🟢 Safe     | Attendance comfortably above threshold |
| 🟡 Warning  | Attendance getting close to danger     |
| 🔴 Critical | High shortage risk                     |

---

## 🤖 AI Attendance Insights

Integrated AI helps students understand their attendance condition intelligently.

Examples:

* “You can bunk 2 more DBMS lectures safely.”
* “Your Mathematics attendance is critically low.”
* “Attend next 5 Physics classes continuously.”

AI-generated guidance makes the app feel like a personal academic assistant.

---

## 📷 OCR Attendance Scanner

Students can upload attendance screenshots/PDFs and automatically extract attendance data using OCR.

This removes the need to manually enter:

* present classes,
* total classes,
* percentages,
* subject names.

---

## 📱 Fully Responsive UI

Optimized for:

* Mobile devices
* Tablets
* Laptops
* Desktop screens

Students can quickly check attendance between lectures without friction.

# 🛠️ Installation

## Clone Repository

```bash
git clone https://github.com/akash0953-py/JewelTrack-App.git
```

## Navigate Into Project

```bash
cd bunksafe
```

## Install Dependencies

```bash
npm install
```

## Setup Environment Variables

Create a `.env.local` file:

```env
GROQ_API_KEY=your_api_key_here
```

---

# ▶️ Run Locally

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```

# 📦 Build For Production

```bash
npm run build
```
